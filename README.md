# AutoLeak

------------

> [!IMPORTANT]
> This branch is not stable. Please use the [beta](https://github.com/FortniteFevers/AutoLeak/tree/beta) branch instead.

------------

**AutoLeak** is a free, open-source, terminal-based utility designed for the Fortnite community. It automates the process of datamining and generating leaks from new game updates/patches.

## Features

- Automatically extracts and processes unreleased Fortnite content (cosmetics, emotes, weapons, etc.) after patches.
- Generates clean, high-quality **cosmetic icons**, weapon icons, and Item Shop-style preview images.
- Bonus tools including custom icon generation and AES key handling for decrypting game files.
- Highly configurable via a `settings.json` file.
- Supports different icon styles (including newer BenBot-powered icons).

It builds on community tools like Fortnite-API, BenBot, Neonite, and others to simplify what used to be a complex manual datamining process.

## Requirements

- Python 3.9 or higher
- Fortnite game files (for local datamining)
- Dependencies installed via `pip install -r requirements.txt` (or the provided `INSTALL PACKAGES.bat`)

## Quick Start

1. Clone or download the repository.
2. Install required packages.
3. Configure your `settings.json` (API choice, icon type, output paths, etc.).
4. Run the main script in your terminal.

For detailed setup instructions, error troubleshooting, and usage examples, check the repository's Wiki or the YouTube tutorials available in the community.
